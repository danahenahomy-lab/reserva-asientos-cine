# reserva-asientos-cine
# Reserva de Asientos en una Sala de Cine

Este programa en Python simula la reserva de asientos en una sala de cine utilizando una matriz de 3 filas y 4 columnas.

## Funcionamiento

El programa realiza las siguientes acciones:
1. Crea una matriz 3x4 donde todos los asientos están libres (valor 0).
2. Solicita al usuario ingresar la fila y la columna del asiento que desea reservar.
3. Marca el asiento seleccionado como reservado (valor 1).
4. Muestra en pantalla el estado de todos los asientos de la sala.

## Significado de los valores

- 0 = Asiento libre
- 1 = Asiento reservado

## Lenguaje utilizado

Python
